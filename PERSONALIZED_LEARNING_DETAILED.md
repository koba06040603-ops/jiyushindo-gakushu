## 📚 Part 1: 学習スタイルが学習カードをどう変えるか

### 🎨 視覚優位型（Visual）の児童への最適化

#### **Before（通常の学習カード）**
```
【問題】
分数の足し算をしましょう。

1/2 + 1/3 = ?

まず通分をします。
分母の最小公倍数を求めて、
両方の分数を同じ分母に揃えます。
```

#### **After（視覚優位型に最適化）**
```
【問題】 分数の足し算をしましょう 🎨

┌─────────────────────────────────────┐
│  🍕 ピザで考えよう！                │
│                                     │
│  1/2 のピザ    1/3 のピザ          │
│  ████░░        ███░░░              │
│  ████░░   +    ███░░░   = ?       │
│  (半分)        (3分の1)            │
└─────────────────────────────────────┘

📊 ステップを図で見てみよう：

ステップ1️⃣ 通分する
┌──────────────────────┐
│ 1/2  → 🔵🔵🔵░░░ (3/6)│
│ 1/3  → 🟢🟢░░░░ (2/6)│
└──────────────────────┘

ステップ2️⃣ 足し算する
┌──────────────────────┐
│ 3/6 + 2/6 = 5/6     │
│ 🔵🔵🔵🟢🟢░ (6個中5個)│
└──────────────────────┘

💡 カラフルなヒント：
分母（下の数）を🔴で、
分子（上の数）を🔵で考えてみよう！
```

// 学習カードを個別最適化して表示
async function loadPersonalizedCard(cardId: number, studentId: number) {
  // 1. 学生のプロファイルを取得
  const profile = await fetchStudentProfile(studentId)
  
  // 2. 元のカードデータを取得
  const card = await fetchCard(cardId)
  
  // 3. プロファイルに基づいてカスタマイズ
  const customized = applyPersonalization(card, profile)
  
  // 4. 最適化されたカードを表示
  renderCard(customized)
}

function applyPersonalization(card, profile) {
  const customized = { ...card }
  
  // 学習スタイルに応じたカスタマイズ
  if (profile.learningStyle.primary === 'visual') {
    customized.images = enhanceImages(card.images)
    customized.diagrams = addDiagrams(card.content)
    customized.colors = applyColorCoding(card.content)
  }
  
  if (profile.learningStyle.primary === 'auditory') {
    customized.ttsEnabled = true
    customized.audioGuides = generateAudioGuides(card.content)
    customized.dialogPrompts = addDialogueElements(card.content)
  }
  
  if (profile.learningStyle.primary === 'kinesthetic') {
    customized.interactiveElements = addInteractiveTools(card.content)
    customized.practiceProblems = increasePracticeCount(card.problems)
    customized.gamification = addGameElements(card.content)
  }
  
  // 弱点に応じた特別サポート
  if (profile.weakAreas.includes('文章題')) {
    customized.textProblems = convertToVisual(card.textProblems)
    customized.supportLevel = 'high'
  }
  
  // 時間帯に応じた調整
  const currentTime = new Date().getHours()
  if (!profile.isOptimalTime(currentTime)) {
    customized.difficulty = 'easier'  // 非最適時間は易しめに
    customized.newContent = false     // 復習中心
  }
  
  return customized
}
