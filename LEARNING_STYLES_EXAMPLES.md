# 学習スタイル別カード設計例

## 📘 同じ単元での3つのアプローチ

### テーマ: 小学3年 算数「かけ算の筆算」

---

## 1️⃣ 視覚優位タイプ（Visual Learner）

### **カード特徴**
- 図解・イラスト中心
- 色分けされた説明
- ステップごとの視覚的表示
- マンガ・フローチャート

### **カード例: 2桁×1桁の筆算**

```
【問題】23 × 4 を筆算で計算しましょう

【視覚的説明】
┌─────────────────────┐
│  図解：筆算の手順     │
│                       │
│    2 3  ←── 被乗数    │
│  ×  4  ←── 乗数      │
│  ─────                │
│  ① 一の位: 3×4=12    │
│     ↓                 │
│     2を書く、1を繰り上げ │
│                       │
│  ② 十の位: 2×4=8     │
│     ↓                 │
│     繰り上がり1を足す  │
│     8+1=9             │
│                       │
│    2 3                │
│  ×  4                │
│  ─────                │
│    9 2  ←── 答え     │
└─────────────────────┘

【色分けポイント】
🔴 一の位の計算: 3×4=12
🔵 十の位の計算: 2×4+1=9
🟢 繰り上がり: 1

【ビジュアルヒント】
💡 ヒント1: 図を見て、赤→青の順番で計算
💡 ヒント2: 色の違いに注目！同じ色同士を計算
💡 ヒント3: フローチャートをたどってみよう
```

**補助教材**:
- 📊 筆算の手順フローチャート
- 🎨 色分けされた位取り表
- 📺 アニメーション動画
- 🖼️ 実例イラスト集

## 2️⃣ 聴覚優位タイプ（Auditory Learner）

### **カード特徴**
- 音声説明・読み上げ
- リズム・語呂合わせ
- 対話形式の問題
- 音声ガイド付き

### **カード例: 2桁×1桁の筆算**

```
【問題】23 × 4 を筆算で計算しましょう

【音声ガイド付き説明】🔊
「それでは、いっしょに声に出して計算してみましょう」

ステップ1️⃣（声に出して）
「一の位から計算します。さん かける よん は...？」
→ 「じゅうに！」
「そうですね。じゅうには、2を書いて、1を繰り上げます」

ステップ2️⃣（声に出して）
「次は十の位。に かける よん は...？」
→ 「はち！」
「正解！そして繰り上がりの1を足すと...？」
→ 「きゅう！」

【リズムで覚える】🎵
「いちのくらい、さん かける よん♪
 じゅうに できて、に を かく♪
 じゅうのくらい、に かける よん♪
 はち たす いち で、きゅう になる♪」

【対話式ヒント】💬
💡 ヒント1: 先生「一の位は何×何？」→ あなた「3×4！」
💡 ヒント2: 先生「繰り上がりは？」→ あなた「1！」
💡 ヒント3: 先生「十の位は何になる？」→ あなた「8+1=9！」
```

**補助教材**:
- 🔊 音声読み上げ機能
- 🎵 リズム暗唱音源
- 🎤 音声入力で答える
- 💬 AI対話練習

## 3️⃣ 体感優位タイプ（Kinesthetic Learner）

### **カード特徴**
- 実際に手を動かす
- 具体物・教具を使う
- 身体を使った活動
- 実生活とのつながり

### **カード例: 2桁×1桁の筆算**

```
【問題】23 × 4 を筆算で計算しましょう

【手を動かして学ぼう】✋
準備するもの:
□ ノート
□ えんぴつ
□ 位取りシート（ダウンロード可）

【体験型アクティビティ】

🎯 アクティビティ1: 実際に書いてみよう
┌─────────────────┐
│ ①まず、ノートに筆算の枠を書く │
│ ②23と4を正しい位置に書く    │
│ ③指で計算する順番をなぞる   │
│ ④一の位から実際に計算      │
└─────────────────┘

🧮 アクティビティ2: おはじきで確認
「23個のおはじきを4つのグループに分けて、
 全部で何個？」
→ 実際に並べて数えてみよう！

🏃 アクティビティ3: 歩いて計算
1. 一の位の位置まで歩く（3×4を計算）
2. 繰り上がりカードを持って十の位へ移動
3. 十の位で計算（2×4+1）
4. ゴール！答えが92

【実生活とのつながり】🌍
「お店で23円のあめを4個買うと？」
→ レジで実際に計算してみよう！

【体感型ヒント】💪
💡 ヒント1: 位取りシートに指を置いて順番を確認
💡 ヒント2: 繰り上がりカードを実際に動かしてみよう
💡 ヒント3: 友達とペアで計算ゲーム！
```

**補助教材**:
- 📐 位取りシート（印刷用PDF）
- 🧮 デジタルおはじき
- 🎮 インタラクティブゲーム
- 📦 実物教具キット

## 📊 学習スタイル別の効果
