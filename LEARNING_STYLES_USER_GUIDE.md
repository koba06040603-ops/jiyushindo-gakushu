# 🎨 学習スタイル別サポート機能 - 使い方ガイド

## 📚 概要

このシステムでは、**視覚優位・聴覚優位・体感優位**の3つの学習スタイルに応じた個別最適化サポートを提供できます。

## 🎯 学習スタイルとは？

### 👁️ **視覚優位（Visual Learners）**
- **特徴**: 図やイラスト、色分けなどで理解が深まる
- **得意**: グラフ、図解、マインドマップ、ビジュアル資料
- **支援例**: 
  - カラフルな図表
  - イラスト付き説明
  - 視覚的な手順書

### 👂 **聴覚優位（Auditory Learners）**
- **特徴**: 音読や説明を聞いて理解が深まる
- **得意**: 音読、リズム、語呂合わせ、対話
- **支援例**:
  - 音読活動
  - リズムで覚える
  - 語呂合わせ

### 🤸 **体感優位（Kinesthetic Learners）**
- **特徴**: 実際に触ったり動いたりして理解が深まる
- **得意**: 実験、身体活動、具体物操作
- **支援例**:
  - 実物を使った活動
  - 身体を動かす学習
  - 実験・体験活動

## 🚀 教師向け：学習スタイル機能の使い方

### **ステップ1: カード詳細を開く**
1. 学習カリキュラムページで任意のカードをクリック
2. カード詳細モーダルが表示される

### **ステップ2: 学習スタイルタブを開く**
1. タブナビゲーションから「**🎨 学習スタイル**」をクリック
2. 3つの学習スタイルセクションが表示される

### **ステップ3: AI提案を使う（推奨）**
1. 「**AI提案**」ボタンをクリック
2. 約5-10秒でAIが各学習スタイルに応じた支援内容を自動生成
3. 生成内容:
   - 支援の説明
   - 必要な教材リスト
   - 具体的な活動例
   - 指導上の留意点

### **ステップ4: 手動編集（カスタマイズ）**
各学習スタイルセクションの「**編集**」ボタンをクリックして、内容をカスタマイズできます。

## 📖 使用例

### **例: 小学3年算数「かけ算の筆算」**

#### 👁️ 視覚優位サポート
```
【支援内容】
位取りが一目で分かるマス目シートを使用。色別ペン（十の位:青、一の位:赤）で数字を書き分けることで、視覚的に計算の流れを理解しやすくします。

【必要な教材】
- マス目シート（1cm×1cm方眼）
- 色ペン（青・赤）
- 位取り表

【活動例】
- 色ペンで十の位と一の位を書き分ける
- マス目に沿って数字を整列させる
- 図解された計算手順を見ながら練習
```

#### 👂 聴覚優位サポート
```
【支援内容】
計算手順を声に出して確認。「3×7=21、1書いて2くりあげ」とリズムよく唱えながら計算することで、手順を音で記憶します。

【必要な教材】
- 音読用計算手順カード
- リズム練習用音声

【活動例】
- ペアで計算手順を説明し合う
- リズムに乗せて計算手順を唱える
- 九九を声に出して確認
```

#### 🤸 体感優位サポート
```
【支援内容】
具体物（ブロック10個×3セット）を実際に並べて操作。手を動かしながら「30個の中から何個ずつ取れるか」を体験的に理解します。

【必要な教材】
- ブロック（30個以上）
- 束ねる輪ゴム
- トレー（整理用）

【活動例】
- ブロックを10個ずつ束ねる
- 実際に配って分ける
- 身体を使って数を数える（ジャンプしながら九九）
```

## 💡 ベストプラクティス

### ✅ **推奨する使い方**
1. **まずAI提案を使う**: 基本的なアイデアを素早く生成
2. **子どもの実態に合わせて編集**: クラスの状況に応じてカスタマイズ
3. **3つのスタイル全てを用意**: 子どもが自分に合った方法を選べる
4. **実践後に改善**: 効果を見ながら内容を更新

### ⚠️ **注意点**
- 全ての子どもが1つの学習スタイルだけではない（複合型が多い）
- 最初は3つ全てのサポートを提示し、子どもに選ばせる
- 効果を観察しながら、個々の子どもに最適なスタイルを見極める

## 🔧 技術仕様

### **APIエンドポイント**

#### 1. 学習スタイル提案生成
```
POST /api/card/:cardId/suggest-learning-styles
```
- Gemini APIを使用してAI提案を生成
- カード内容に基づいて3つの学習スタイル別サポートを自動生成

#### 2. カード更新
```
PUT /api/card/:cardId
```
- 学習スタイルフィールドを含むカード情報を更新
- 更新可能フィールド:
  - `visual_support`: 視覚優位サポート
  - `auditory_support`: 聴覚優位サポート
  - `kinesthetic_support`: 体感優位サポート
  - `learning_style_notes`: 指導上の留意点

### **データベーススキーマ**
```sql
ALTER TABLE learning_cards ADD COLUMN visual_support TEXT;
ALTER TABLE learning_cards ADD COLUMN auditory_support TEXT;
ALTER TABLE learning_cards ADD COLUMN kinesthetic_support TEXT;
ALTER TABLE learning_cards ADD COLUMN learning_style_notes TEXT;
```
